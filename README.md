# UNREAD

UNREAD is a lightweight Flashcard-style application that helps users retain information long-term by saving and organizing learning resources (links, notes, Drive files, etc.) and creating shareable flashcards. Users can schedule revisions using common spaced-repetition rules (for example the 1-4-7 rule) or custom dates.

Key features
- Save and organize resources: links, notes, and file references
- Create, edit and delete flashcards
- Make flashcards public so others can discover and reuse them
- Schedule revisions using the 1-4-7 rule or custom dates
- Authentication with JWT and OAuth2 (client)
